# Limited Dry-Run Boundaries

Dataset v0.2.5 is a two-case methodological coding dry run.

## What This Is

- A limited reconstruction coding pass for `ABS-001` and `ABS-002`.
- A test of whether the v0.2 abstraction and coding workflow produces interpretable outputs.
- A source-distance and safety-preserving coding exercise.

## What This Is Not

- Not full Dataset v0.2 coding.
- Not clinical validation.
- Not clinical guidance.
- Not a real-world estimate of transformation frequency.
- Not approval to abstract the remaining 8 candidates.
- Not approval to modify `ABS-001.md` or `ABS-002.md`.

## Inputs Allowed

Allowed inputs:

- `../case-abstractions/ABS-001.md`
- `../case-abstractions/ABS-002.md`
- local coding framework;
- reconstruction protocol;
- evaluation rubric;
- compression versus uncertainty-loss rule.

Disallowed inputs:

- source case text;
- copied published case-report passages;
- source images, figures, tables, or captions;
- patient-identifying information;
- new clinical interpretation beyond the abstractions.

## Hold Condition

The remaining 8 candidates remain on hold until post-coding review decides whether the method is safe and useful enough to continue.
