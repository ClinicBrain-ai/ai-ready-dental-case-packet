## 5. Are Any Sources Methodologically Weak?

CAND-013 and CAND-015 are methodologically weak for the current pilot. CAND-013 is more therapy-program oriented, while CAND-015 lacks a single reconstructable case transition.

CAND-011 and CAND-020 may be useful later but are weaker for the first 10-case abstraction set.

## 6. What Safeguards Should Be Used During Abstraction?

Use these safeguards:

- write artifact pairs from scratch using research-team wording;
- do not copy source sentences, paragraphs, tables, figure captions, or image descriptions;
- omit names, institutions, precise dates, exact ages when unnecessary, locations, and rare identifying sequences;
- generalize clinical chronology to only what is needed for the cognitive transition;
- keep source citation metadata separate from generated artifacts;
- include the required non-clinical-use label on every pair;
- perform a second-review copied-text and identifiability check before coding.

## 7. Should a Human Second Reviewer Verify the Final Abstraction Before Coding?

Yes. A human second reviewer should verify every abstraction before coding. The reviewer should check:

- no copied source narrative;
- no patient-identifying information;
- no source image or table reproduction;
- no clinical advice language;
- enough initial and downstream artifact structure to support reconstruction;
- clear labeling as educational abstraction.

## 8. Should Research Note 002 Wait Until Two-Coder Coding Is Complete?

Yes. Research Note 002 should wait until the two-coder workflow is complete or until a documented stop condition is reached. Publishing a note before two-coder coding would risk overstating the value of Dataset v0.2 based only on source selection.

## Readiness Conclusion

Dataset v0.2 is ready for limited educational abstraction after accepting the revised shortlist and applying the safeguards above. It is not ready for coding, quantitative comparison, or research-note conclusions.
