# Case Abstraction Protocol

This protocol defines how to convert a published educational case into a reconstruction-ready artifact pair without copying source text.

The abstraction must not be represented as real clinical documentation.

## Steps

1. Read source case.

   Review the public educational case for structure, not for clinical reuse. Do not copy full passages.

2. Identify initial clinical presentation or referral-like artifact.

   Determine whether the source includes an initial complaint, referral question, primary assessment, or early clinician concern.

3. Identify downstream assessment or final plan.

   Determine whether the source includes specialist interpretation, final diagnosis, treatment plan, follow-up plan, or explanatory rationale.

4. Create a short paraphrased educational abstraction.

   Convert the case into a concise referral-like artifact and consultation-like artifact using original wording from the research team, not copied source text.

5. Remove or avoid identifiable details.

   Exclude names, exact dates, institution names, locations, rare identifying sequences, images, and other details not needed for the cognition transition.

6. Preserve only the cognitive transition relevant to the study.

   Keep the elements needed to study referral concern, uncertainty, downstream interpretation, rationale, and drift. Omit nonessential narrative detail.

7. Record citation/source metadata separately.

   Use the case index to record citation or URL metadata. Keep source metadata separate from the paraphrased artifact pair.

8. Label the pair as educational abstraction, not original clinical documentation.

   Every pair should state that it is a paraphrased educational abstraction based on a public source pattern.

9. Flag uncertainty about whether the abstraction reflects the original reasoning.

   Record whether the source was retrospective, didactic, incomplete, or ambiguous.

## Required Label

Each abstracted pair should include:

> This is a paraphrased educational abstraction for methodology testing. It is not original clinical documentation and should not be used for patient care.

## Prohibited Content

- Full copied case text.
- Paywalled source text.
- Patient-identifying details.
- Source images, tables, or figures unless reuse rights are explicit and documented.
- Clinical recommendations generated by the abstractor.
