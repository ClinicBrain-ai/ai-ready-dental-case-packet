# DCS White Paper

This directory contains the historical standards-oriented white paper for the Dental Context Specification (DCS).

In the current CCTL framing, DCS is no longer the central project. It is preserved here as an origin artifact and a candidate cognitive provenance representation layer for Human-AI Clinical Cognition research.

DCS may still be useful as a minimal claim provenance layer for studying semantic accountability across dental agent handoffs. It should be read as a downstream representation experiment rather than the primary identity of the repository.
