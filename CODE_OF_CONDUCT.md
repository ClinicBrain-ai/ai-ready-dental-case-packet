# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, professional, and harassment-free for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Prioritize patient privacy and safety.
- Keep clinical claims conservative.
- Welcome contributors from dentistry, software, standards, privacy, and AI infrastructure.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination.
- Sharing real patient data without authorization.
- Encouraging diagnostic or treatment recommendation features outside the project scope.
- Publishing private security or PHI issues instead of reporting them responsibly.

## Enforcement

Maintainers may remove comments, issues, pull requests, or contributors that violate this code of conduct.
