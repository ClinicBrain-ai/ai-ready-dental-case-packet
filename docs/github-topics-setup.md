# GitHub Topics Setup

GitHub topics must be applied manually by a repository maintainer.

## Recommended Topics

- `mcp`
- `model-context-protocol`
- `ai-agents`
- `agent-tools`
- `dental-ai`
- `cbct`
- `dicom`
- `healthcare-ai`
- `llm-tools`
- `context-layer`
- `agent-infrastructure`

## Manual GitHub UI Steps

1. Open the repository:

   https://github.com/ClinicBrain-ai/ai-ready-dental-case-packet

2. On the repository homepage, find the About panel on the right side.

3. Click the gear icon next to About.

4. In the Topics field, add:

   ```text
   mcp
   model-context-protocol
   ai-agents
   agent-tools
   dental-ai
   cbct
   dicom
   healthcare-ai
   llm-tools
   context-layer
   agent-infrastructure
   ```

5. Confirm the repository description is:

   ```text
   Building the Dental Context Layer for AI Agents.
   ```

6. Confirm the website field points to the README, docs, or latest release if desired.

7. Click Save changes.

## Why These Topics Matter

These topics make the repository easier to discover through GitHub search, MCP ecosystem search, package manager indexing, AI coding agents, and future agent marketplaces.
