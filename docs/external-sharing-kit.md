# External Sharing Kit

## One-Sentence Description

Clinical Cognition Transformation Lab (CCTL) studies how clinical cognition transforms in distributed human-AI healthcare systems.

## Short Description

CCTL is a research program studying how clinical cognition changes when healthcare decisions are shaped by clinicians, AI systems, documentation artifacts, workflows, institutions, patients, and caregivers. It exists because clinical reasoning is increasingly distributed across human and non-human participants rather than contained within a single clinician's mind. The repository currently contains conceptual framing, early paper drafts, pilot methods, synthetic feasibility work, and a closed Dataset v0.2 methodology cycle. It is educational, methodology-focused, and not clinical guidance.

## Medium Description

Clinical Cognition Transformation Lab (CCTL) evolved from dental AI infrastructure work, including the AI-ready Dental Case Packet and Dental Context Specification, into a broader research program on Human-AI Clinical Cognition. The project moved from questions about AI-ready clinical data and context representation toward the larger problem of how clinical cognition is preserved, compressed, delegated, reconstructed, or transformed as it moves through distributed healthcare systems.

CCTL's current research focus is clinical cognition transformation in distributed human-AI healthcare systems. The repository includes a manifesto, research agenda, paper drafts, pilot study methods, Research Note 001, and a closed Dataset v0.2 cycle. Dataset v0.2.8 was closed after limited abstraction, limited coding, and human post-coding review; Dataset v0.2.9 consolidated that status.
