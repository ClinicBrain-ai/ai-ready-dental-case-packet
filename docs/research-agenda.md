# Research Agenda

CCTL studies how clinical cognition transforms when healthcare becomes a distributed human-AI cognitive system.

## 1. Clinical Cognition Transformation

This program studies how cognition changes across healthcare systems.

Phenomena include:

- diagnostic reframing;
- recommendation drift;
- uncertainty compression;
- silent diagnostic replacement;
- mechanism substitution;
- cognitive delegation;
- diagnostic deferral;
- patient-facing simplification.

The central question is how clinical meaning, uncertainty, responsibility, and action change as cognition moves between clinicians, AI systems, artifacts, workflows, organizations, patients, and caregivers.

## 2. Cognitive Reconstruction

This program studies whether cognition can be reconstructed from downstream artifacts.

Relevant artifacts include:

- referral letters;
- consultation notes;
- treatment plans;
- clinical documentation;
- longitudinal records.

Key question:

How much cognition survives after the original reasoning event disappears?

Cognitive Reconstruction treats downstream artifacts as partial traces of prior cognition. It asks what can be inferred, what is missing, what is distorted, and what kinds of reasoning become unrecoverable.

## 3. Human-AI Co-Cognition

This program studies hybrid cognitive systems involving humans and AI.

The focus is not only whether AI improves performance. The focus is how cognition is jointly produced when clinicians, models, interfaces, records, guidelines, and patients participate in interpretation and decision-making.

Questions include:

- When does AI extend clinician cognition?
- When does it substitute for clinician cognition?
- When does it redirect attention or reframe the case?
- When do humans and AI produce a cognitive output neither would have produced alone?
- How should responsibility be understood when cognition is distributed?

## 4. Cognitive Provenance

This program studies how cognitive pathways can be represented, traced, audited, and understood.

Cognitive provenance includes the path by which clinical claims, interpretations, uncertainties, recommendations, and responsibilities move through a system.

DCS is one candidate representation layer for this program. It may help encode clinical context, reasoning traces, uncertainty, and workflow transitions, but it is not the central research question.

## 5. Longitudinal Cognitive Change

This program studies how clinician cognition evolves as AI becomes embedded into daily practice.

Questions include:

- Which cognitive abilities are amplified?
- Which are delegated?
- Which become obsolete?
- Which new forms of expertise emerge?
- How do experts themselves change?

The program treats expertise as dynamic. It asks how clinical training, attention, memory, pattern recognition, explanation, uncertainty handling, and accountability change over time in AI-embedded care.
